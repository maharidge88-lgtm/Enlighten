# Enlighten
A protocol for finding enlightenment through accelerated learning skills from various areas of study, sound frequency exposure, breathing techniques that enhance focus, concentration, and memory combined with awareness practices to help create a balanced, solid coherence in learning capabilities

## Files
- [Enlightenment_Protocol.md](Enlightenment_Protocol.md): The detailed step-by-step protocol with visualizations and scientific explanations.
- [app.py](app.py): Companion Streamlit app for personalized practice plans and science explanations.
- [requirements.txt](requirements.txt): Dependencies for the app.

## Running the Companion App
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`
3. Open http://localhost:8501 in your browser.

The app helps create individualized plans based on your purpose and desired results, explains the science behind each practice, and guides you through sessions.
